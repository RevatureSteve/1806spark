how to create an angular routing?
    1. create an app-routing.module.ts
    2. export the class
    3. create a route array, an imported angular class
    4. provide the route array path {}s that require the 2 properties
            path and component
    5. provide the module decorator: 
    6. provide the @NgModule decorator: imports / exports
        e.g. @NgModule ({

            imports: [RouterModule.forRoot(routes)],
            export: [RouterModule]

        })
    7. add the <router-outlet></router-outlet> directive to the place 
            you want the pages to show
    8. add to app.module.ts to the imports array
        imports: [
            BrowserModule,
            AppRoutingModule
        ]
    9. update navbar with routerLink not the href
