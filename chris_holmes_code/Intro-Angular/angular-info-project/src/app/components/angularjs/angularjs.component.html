AngularJS is a JavaScript framework, Angular is a TypeScript framework (which transpiles into JavaScript).
Angular replaces AngularJS Controllers with Components.
Angular's performance improves because of the Component's lifecycle (and other reasons).
AngularJS only has two forms of databinding (one-way and two-way) and it actually uses directives (ng-bind, ng-model) for them.
The amount of directives is reduced in Angular because of the extra ways of data binding and components.
AngularJS follows the MVC architecture strictly, Angular doesn't follow any specific design pattern.
There are more I invite you to check AngularJS code and investigate more differences.
