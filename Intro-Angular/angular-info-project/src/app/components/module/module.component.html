{{title}}
Makes Angular applications modular.
They are normally described in a .module.ts file.
They are defined with the {{subTitleNgModule}} decorator.
{{subTitleNgModule}}
The {{subTitleNgModule}} decorator receives metadata.
The metadata tells Angular how to compile and launch the app.
Components and Pipes go in the declarations section.
Other modules go in the imports section.
Services go in the providers section.
In the bootstrap we define which component loads first.
