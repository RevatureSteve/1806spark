Node.js

is a javascript server software, can handle requests and send back responses.
(not recommended, requires too much work.) - use express in addition.

cmd: node (w/o any other argument) turns your terminal into a javascript interpreter.
ultimate command to stop process: ctrl + c

express (JavaScript Package)
a javascript framework
    used with node.js to make it easy to handle requests and
    sent back responses. (remove back responses)

Node Package Manager: allows to quickly download JavaScript packages like express.

cmd = npm

turn our tampa gems project into a node project
cmd: npm init

npm init creates a package.json file which is the
meta data of our project...
